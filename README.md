# SwarmIntelliScheduling
SwarmIntelliScheduling(SIS) is an open-source software that integrates order, process, and equipment information into an Access database for unified query and management. It employs an integrated swarm intelligence optimization algorithm to solve the Flexible Job Shop Scheduling Problem (FJSP) in production scheduling.

# Quick Start
1)Create an ODBC data source named SISAPS_EN on Windows to connect to the database SISDatabase.accdb.
2)Run app_SISAPS.mlapp.
